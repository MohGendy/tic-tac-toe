# Build Instructions

## To Build Use Next Commands

```sh
cd ./build  
make clean  
cmake .. -G "MinGW Makefiles"  
make
```
## after building run the excutable
```
my_program.exe
```
# you need to clone google test to run the tests
```sh
git clone https://github.com/google/googletest.git
```
## to run tests
```sh
runTests.exe
```
